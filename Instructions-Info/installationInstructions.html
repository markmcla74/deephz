Install DeepHz on iPhone

  1. Open Safari (Important: it won’t work properly in Chrome or other browsers)
  2. Go to this link: https://markmcla74.github.io/deephz/
  3. Tap the Share button (square with an arrow at the bottom of the screen)
  4. Scroll down and tap “Add to Home Screen”
  5. Tap “Add” (top right)
  • DONE! You’ll now see a DeepHz icon on your home screen
  • Tap it anytime to open it like an app

Install DeepHz on Android

  1. Open Google Chrome (or another browser like Samsung Internet)
  2. Go to this link: https://markmcla74.github.io/deephz/
  3. Wait a few seconds. You might see a popup: “Install app”
  4. If you see it → tap Install

  • Tap it anytime to open it like an app

How to Keep your iPhone Awake using Guided Access

One-time setup:

  • Open Settings
  • Tap Accessibility
  • Scroll down and tap Guided Access
  • Turn Guided Access ON
  • Tap Passcode Settings and set a passcode

Start Guided Access:

  • Open the DeepHz app
  • Triple-click the Side button
  • Tap Options (bottom left)
  • Set Auto-Lock to Never
  • Tap Start

To exit Guided Access:

  • Triple-click the Side button again
  • Enter your passcode
  • Tap End

How to keep your Android phone awake

  • Install an app that will keep your screen on. For example, install the Screen Alive App from the Google Play Store.
